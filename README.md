# Python To-Do List App

A desktop To-Do List application built using Python and Tkinter with local storage using JSON.

## Features

- Add new tasks
- Edit existing tasks
- Delete selected tasks
- Mark tasks as complete or incomplete
- Set task priority as High, Medium, or Low
- Save tasks locally using JSON
- Interactive GUI built with Tkinter
- Input validation using warning messages

## Technologies Used

- Python
- Tkinter
- JSON
